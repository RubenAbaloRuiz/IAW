ACT 03 : JAVASCRIPT - Entering Data.


To introduce information using a keyboard we can use the function prompt. Every time we need to enter data using this function, a dialog window with an input field appears and we can type the information. To see how it works let's study the following example:




  

     
 <html>
 <head></head>
 <body>
 <script type="text/javascript">
 var name;
 var age;
 name=prompt('Enter your name:','');
 age=prompt('Enter your age:','');
 document.write('Hello ');
 document.write(name);
 document.write(', you are ');
 document.write(age);
 document.write(' years old.');
 </script>
 </body>
 </html>

  • Syntax of the prompt function is:



  

    
      
<variable_name> = prompt(<message_to_show>,<initial_value_to_show>);


The prompt function has two parameters: the first one is the message and the second one is the initial value to show in the dialog box.




Exercises.

a) Write a script that allows us to enter the user name and his/her email using the keyboard. After all print every data in the web page.
